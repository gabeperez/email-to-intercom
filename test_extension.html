🔍 What to Check

✅ Expected Results:

  • Email addresses should be clickable (blue and underlined)
  • Clicking an email should open the email composer panel
  • Messages should preserve line breaks and spacing
  • Paragraphs should be properly separated
  • Special characters should be properly escaped
  • Emails should send successfully to Intercom

❌ If Something Goes Wrong:

  • Check browser console for errors (F12 → Console)
  • Verify Intercom credentials are configured correctly
  • Test connection in extension settings
  • Check if extension has proper permissions
